## Prerequisites
- Python 2.6 or greater
- pip package management tool
- A Google Account

## Install Setup
```
pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib
```

## Cron Config
```
crontab -e 
0 22 * * * cd [hkn-pacman dir] && python pacman.py
```
